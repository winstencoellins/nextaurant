# Nextaurant

## Technologies

Technologies used in this project:
- Next Js (version 14)
- Tailwind CSS
- npm (node package manager)

## Installation / Setup

Clone this repository to your local machine

```
git clone https://github.com/winstencoellins/nextaurant.git
```

<br />

Install node dependencies

```
npm install
```

<br />

Run the code

```
npm run dev
```
